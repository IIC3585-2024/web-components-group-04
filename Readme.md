## Web components

Este proyecto incluye dos componentes de ejemplo: una lista de tareas (todo-list) y un ítem para vender (sell-item).

### Ejecución
1. Instala las dependecias:
   ```bash
   npm install 
   ```
2. Ejecuta:
    ```bash
   npm start 
   ```
